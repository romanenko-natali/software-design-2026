# 3. User Stories

> Format: _"As a [role], I want [action], so that [value]."_
> Each User Story must have at least 3 acceptance criteria.
> Group stories by Use Case for clarity.

## Priority Legend

| Priority | Meaning |
|----------|---------|
| 🔴 High | Critical — system does not work without this |
| 🟡 Medium | Important, but a workaround exists |
| 🟢 Low | Nice to have if time allows |
